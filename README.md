# Chikuwa-s-trip-and-magic
結構大作になる気がするデータパック。 旅と魔法を楽しくすることをモットーに作っている。

旅では、新しい武器や料理、構造物にボスや新たなMOBも出てくる。 魔法では、主に炎,氷,水,雷,自然,虚無の魔法を追加して、魔法の杖と魔導書、魔法の武器を追加する。
